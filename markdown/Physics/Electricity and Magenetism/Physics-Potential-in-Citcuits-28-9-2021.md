# Circuits

## Deriving Equation for Potential Divider

$$I = \frac{V}{R}$$

I -> same

$$I_1 = I_2$$

<br/>


$$\frac{V_1}{R_1} = \frac{V_2}{V_2}$$

<br/>

$$V = V_2 + V_2$$

<br/>

$$V = IR$$

<br/>


$$V = I(R_1 + R_2) \rightarrow I = \frac{V}{R_1 + R_2}$$

<br/>


$$V_1 = I \times R_1$$

<br/>


$$\frac{V_1}{R_1} = \frac{V}{R_1 + R_2}$$

<br/>

$$V_1 = (\frac{R_1}{R_1 + R_2}) \times V$$

<br/>


$$V_2 = (\frac{R_2}{R_1 + R_2}) \times V$$



<br/>
